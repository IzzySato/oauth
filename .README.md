# Node Project
- author: Shizuka (Izzy) Sato

## Technologies
- Node.js
- postgres sql


## Start the postgres

1. 'brew services start postgresql@13'
2. To connect: 'psql -h localhost -p 5432 -U postgres'

## Stop the postgres

1. 'brew services stop postgresql@13'

## How to start the project

1. npm install
2. npm start
3. listening localhost 3001

## How to run test

1. 'npm test'
